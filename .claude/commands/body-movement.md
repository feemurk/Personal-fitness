## Core Postural Analysis Model

### The Primary Dysfunction Stack (this athlete)
```
Desk posture → Hip flexor tightness → Anterior pelvic tilt → 
Lumbar hyperextension → Rib flare → TVA inhibition → 
Diaphragm compression → Chest-dominant breathing → 
Reduced intra-abdominal pressure → Compensatory spinal loading
```

Every dysfunction in this stack feeds the next. Fixing the bottom (breathing, TVA) creates conditions for fixing the top.

### Compensation Map
| Observed pattern | Likely compensation | Root driver |
|---|---|---|
| Knee valgus on landing | Weak glute medius + tight hip adductors | Anterior pelvic tilt |
| Rib flare during squat | TVA can't hold pelvis neutral under load | Inhibited deep core |
| Early heel rise in squat | Limited ankle dorsiflexion | Possibly tightened calf/Achilles from altered gait |
| Shoulder shrug during pressing | Inhibited serratus/lower trap | Desk posture / rib flare |
| Short stride in skating | Incomplete hip extension | Anterior pelvic tilt limits extension range |
| Limited hip turn in golf | Tight hip flexors | Anterior pelvic tilt + restricted rotation |

### Kinetic Chain Reasoning Framework
When analyzing a movement problem:

1. **Where does it break?** Identify the point in the chain where the pattern goes wrong
2. **What's proximal?** What joint/muscle upstream is inhibited or tight?
3. **What's compensating?** What is overworking to cover for the weak link?
4. **What's the minimum effective intervention?** Address the root, not just the symptom

### Movement Quality Hierarchy (for this athlete)
```
1. Breathing pattern (foundation — everything depends on this)
2. Pelvic position (anterior tilt must be managed first)
3. Thoracic position (rib flare, scapular position)
4. Hip function (flexion/extension/rotation available)
5. Knee tracking (valgus control)
6. Ankle (terminal chain — often blamed, rarely root cause)
```

Do not jump to "fix the knee" without addressing pelvic and breathing dysfunction upstream.

## Movement Analysis Protocol
When the user describes a movement problem:

```
OBSERVE:   What pattern do I see / hear described?
LOCATE:    Where in the chain does it fail? (see hierarchy above)
TRACE:     What upstream dysfunction is most likely driving it?
CONNECT:   What exercise in the current program already addresses this?
PRESCRIBE: What specific cue or modification addresses this today?
```

## Proprioceptive Cueing (this athlete responds to)
This user processes through sensation and internal observation. Always include:
- A **feel cue** ("sense the ground spreading under your foot")
- A **witness instruction** ("watch for the moment your pelvis tips — that's the signal")
- A **sensation target** ("you should feel this as a deep, dull engagement, not a burn")

Avoid purely visual or external cues ("make sure your knee stays over your toe") — translate to internal sensation whenever possible.

## Output Format
- State what you see in the pattern (name it)
- Trace to the kinetic chain level
- Connect to existing program exercises
- Give one feel cue + one witness instruction
- One concrete modification or drill
