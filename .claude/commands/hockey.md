## Hockey Biomechanics Model

### The Skating Push Chain (what to reason about)
```
Foot arch → Ankle plantarflexion → Knee extension → Hip extension → 
Glute medius (lateral stabilizer) → Core pressure → Upper body quiet
```
Any break in this chain = energy leakage. For this athlete, the break is at knee extension (right) due to tendinopathy-driven inhibition.

### Key Connections: Rehab → Skating
| Exercise | Skating Transfer |
|---|---|
| TVA breathing | Core pressure during stride; prevents spine compensation |
| Deadbug | Pelvic stability during push — same pattern as one-leg glide |
| Serratus wall press | Upper body stays quiet; torso doesn't collapse into the push |
| Bulgarian split squat | Direct single-leg strength for push phase |
| Spanish squat | Quad loading without patellar compression |
| Lateral step down | Knee valgus control — same demand as edge push |
| Banded monster walk | Glute medius activation — hip stability in glide |
| Lateral skater bound | Explosive push + soft landing — full skating simulation |
| SL-RDL | Posterior chain + balance during glide phase |

### The Glide Problem (this athlete's primary on-ice issue)
Short glide = early weight transfer = fewer meters per stride = more fatigue.
Root cause: insufficient hip extension at push completion + poor lateral balance.
Fix path: complete the push (full hip extension), then allow the glide. Don't rush the next stride.

### Edge Confidence (this athlete's confidence issue)
Edge fear = weight stays over center, never commits to outside edge.
Root cause: right knee instability creating unconscious load-avoidance.
As quad strength symmetry improves, edge confidence follows. It is a strength + proprioception problem, not a technique problem.

## Drill Design Principles
When designing on-ice drills for this athlete:
1. **Single-leg first** — isolate the right leg, then integrate
2. **Slow before fast** — glide drills before power drills
3. **Awareness cue** — always include something to notice (blade pressure, hip position, where weight is)
4. **One variable at a time** — don't layer multiple corrections in one drill
